# Neon Shift

Неоновая браузерная головоломка на поле 7×7: освободи бирюзовое ядро и доведи его до портала справа.

## Запуск

Ничего устанавливать не нужно. Открой `index.html` двойным кликом в современном браузере.

Если браузер ограничивает локальные файлы, можно запустить встроенный сервер Python из папки проекта:

```powershell
python -m http.server 8000
```

После этого открой `http://localhost:8000`.

## Управление

- Потяни блок мышкой или пальцем вдоль его оси.
- Горизонтальные блоки двигаются только влево и вправо, вертикальные — вверх и вниз.
- Кнопки позволяют отменить ход, перезапустить сектор или получить подсказку.
- После выбора блока также работают клавиши-стрелки.

## Как устроены уровни

Каждый сектор генерируется детерминированно по номеру уровня. Генератор создаёт раскладку, а встроенный BFS-решатель проверяет её и вычисляет минимальное число ходов. Непроходимые и слишком простые варианты отбрасываются; с ростом номера сектора увеличиваются число блоков и требуемая глубина решения.

Прогресс, звук и лучшие результаты сохраняются в `localStorage` браузера.

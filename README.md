# Neon Shift

Неоновая браузерная головоломка на поле 7×7: освободи бирюзовое ядро и доведи его до портала справа.

## Запуск

Ничего устанавливать не нужно. Открой `index.html` двойным кликом в современном браузере.

Если браузер ограничивает локальные файлы, можно запустить встроенный сервер Python из папки проекта:

```powershell
python -m http.server 8000
```

После этого открой `http://localhost:8000`.

## Управление

- Потяни блок мышкой или пальцем вдоль его оси.
- Горизонтальные блоки двигаются только влево и вправо, вертикальные — вверх и вниз.
- Кнопки позволяют отменить ход, перезапустить сектор или получить подсказку.
- После выбора блока также работают клавиши-стрелки.

## Как устроены уровни

В игре лежит готовый набор из 50 проверенных секторов в компактном файле `levels.js`. При запуске браузер читает раскладки из этого файла — генерация во время игры не выполняется. Для каждого сектора заранее рассчитано минимальное число ходов.

Скрипт `scripts/generate-level-pack.js` позволяет разработчику заново собрать набор встроенным генератором и BFS-решателем. Для обычной игры запускать его не нужно.

Текущий уровень, пройденная часть кампании, звук и лучшие результаты сохраняются в `localStorage` браузера. После обновления или повторного открытия страницы игра продолжится с сохранённого сектора.
